创建一个文本

div1.appendChild(text1) div1.innerText = '你好' div.textContent = '你好' JS创建的节点 需要加入head或者body中才可以 document.body.appendChild(div) 页面中的元素.appendChild(div)`

改属性

改class：div.className = 'red blue' 改class：div.classList.add('red') 改style：div.style = 'width = 10px; color = blue;' 改style部分：div.style.width = '50px' 大小写：div.style.backgroundColor = 'white' 改data-* 属性：div.dataset.x = 'haha'

遍历div里的每一个元素

travel = (node,fn)=>{
	fn(node)
	if(node.children){
		for(let i = 0; i < node.children.length; i++){
 				travel(node.children[i],fn)
		}
  }
}
